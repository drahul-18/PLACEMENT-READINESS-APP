# KodNest Premium Build System

Design system for B2C product experiences. Calm, intentional, coherent, confident.

## Design Philosophy

- Calm, Intentional, Coherent, Confident
- Not flashy, not loud, not playful, not hackathon-style
- No gradients, no glassmorphism, no neon colors, no animation noise

## Color System (4 colors max)

| Token | Value | Usage |
|-------|-------|-------|
| `--color-bg` | #F7F6F3 | Background |
| `--color-text` | #111111 | Primary text |
| `--color-accent` | #8B0000 | Accent, primary actions |
| `--color-success` | #5a6b54 | Success states |
| `--color-warning` | #8b7355 | Warning states |

## Typography

- **Headings**: Source Serif 4, large, confident
- **Body**: Source Sans 3, 16–18px, line-height 1.6–1.8, max 720px for text blocks

## Spacing Scale

8px, 16px, 24px, 40px, 64px — never use random values.

## Layout Structure

1. **Top Bar**: Project name | Progress (Step X/Y) | Status badge
2. **Context Header**: Serif headline + 1-line subtext
3. **Main Content**: Primary Workspace (70%) + Secondary Panel (30%)
4. **Proof Footer**: Checklist (UI Built, Logic Working, Test Passed, Deployed)

## Components

- **Primary button**: Solid deep red
- **Secondary button**: Outlined
- **Inputs**: Clean borders, clear focus state
- **Cards**: Subtle border, no drop shadows
- **Transitions**: 150–200ms ease-in-out
